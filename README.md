# GifHub Project

GifHub is a mini web project that integrates with the Giphy API to display trending GIFs and allow users to search for their favorites in real time. Built with HTML, CSS, and JavaScript, it demonstrates responsive design, API integration, and dynamic content rendering.

## Features

- Displays trending GIFs on page load
- Live search functionality with instant results
- Responsive gallery layout for different screen sizes
- Error handling for API requests

## Learning Goals

This project is part of my journey into web development, focusing on API usage and front-end interactivity.
